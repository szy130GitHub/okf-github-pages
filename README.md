# OKF GitHub Pages External Link Page

This is the static GitHub Pages overview page for OKF Tools. It explains the Open Knowledge Format workflow, shows real product screenshots, records recent updates, and routes visitors to the main tool site:

https://okftools.com

## Local Verification

```bash
npm test -- tests/github-pages-static.test.ts
```

## Publishing

The `github-pages/` directory can be published as an independent static GitHub Pages repository. The publishing source is the root of the `main` branch.
