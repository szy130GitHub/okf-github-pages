Features

A practical path from sitemap to /okf/index.md

OKF is useful when it gives agents a structured way to discover and traverse your knowledge. OKF Tools combines bundle generation, validation, scoring, and deployment guidance in one workflow.

Generator

OKF Generator

Enter a sitemap.xml URL or a list of page URLs to generate index.md, log.md, concepts/*.md, and a downloadable zip bundle.

Validator

OKF Validator

Check Markdown frontmatter, required type fields, internal links, missing descriptions, and thin content before publishing.

Score

OKF Score

Combine structure, connectivity, and coverage into a 0-100 score that is easy to compare, share, and improve over time.

Deploy

Deployment Guide

Publish generated files under /okf/ and verify that https://site.com/okf/index.md is reachable from the public web.

Who should publish an OKF bundle

Website owners and indie products

Package product pages, pricing pages, help docs, and changelogs into structured Markdown that gives agents a clearer entry point.

SEO and GEO teams

Use OKF Score to audit AI-readable content quality, spot missing descriptions, identify thin concepts, and repair broken Markdown links.

Developers and documentation teams

Turn docs, runbooks, API pages, and knowledge bases into versionable Markdown instead of forcing agents to infer meaning from raw pages.

FAQ

Common questions

How is this GitHub Pages site related to OKF Tools?

This page is a static external-link overview. Live generation, validation, reports, badges, and API-backed workflows remain on https://okftools.com.

Why not deploy the full tool site on GitHub Pages?

The main OKF Tools site uses Next.js API routes to fetch sitemaps, generate zip files, validate bundles, and create reports. GitHub Pages is static hosting, so this page links to the live tool instead.

How is OKF different from llms.txt?

llms.txt is closer to a reading guide for models. OKF is a traversable Markdown knowledge bundle with frontmatter, links, concepts, and citations. They work well together.

Where should I start?

If your site does not have an OKF bundle yet, start with the generator and a sitemap. If you already have Markdown files, start with the validator and review the score and fixes.
